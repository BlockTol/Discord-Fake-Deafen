# FakeDeafen

A Vencord plugin that lets you appear deafened to others while still being able to hear everything!

---

## Features

| Feature | Description |
|---------|-------------|
| **Fake Deafen** | Appear deafened to others while still hearing audio |
| **Context Menu** | Toggle directly from the audio device right-click menu |
| **Custom Keybind** | Set any key combination to quickly toggle fake deafen |
